# Bundled Fonts

## JurassicPark-BL48.ttf

- **Source:** https://www.fontspace.com/jurassic-park-font-f3555
- **License:** Freeware, **Non-Commercial** use only.

This is the *Jurassic Park*-style movie title face, bundled for use as a
hero/display font (see `design/typography.md`, "Large Display Text").

**Important:** Unlike the OFL fonts recommended in `design/typography.md`
(Oswald, JetBrains Mono, Archivo Black), this font is licensed for
**non-commercial use only**. That is compatible with NublarOS being a free,
non-commercial, fan-made parody project (see "Legal and Asset Guidelines" in
the root README), but it means:

- NublarOS must remain non-commercial while this font is bundled.
- Anyone forking NublarOS for a commercial purpose must remove this font and
  substitute a commercially-licensed display face.

The same file is mirrored at
`design-system/src/main/resources/fonts/JurassicPark-BL48.ttf` so the Compose
app can load it at runtime.
